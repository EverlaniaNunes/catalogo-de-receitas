# 🍰 Catálogo de Receitas

Projeto desenvolvido para a disciplina de Desenvolvimento Web.

## 📌 Descrição

O projeto consiste em um catálogo de receitas que consome dados de uma API utilizando a Fetch API. As receitas são exibidas em cards organizados com CSS Grid e estilizados com Tailwind CSS. Também é possível pesquisar receitas pelo nome.

## 🚀 Tecnologias utilizadas

- HTML5
- JavaScript
- Tailwind CSS
- Fetch API
- CSS Grid
- Flexbox

## ✨ Funcionalidades

- Exibe receitas em formato de cards.
- Consome dados de uma API.
- Pesquisa receitas pelo nome.
- Layout responsivo.
